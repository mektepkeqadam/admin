## ✨ Функции

### 👤 Для пациентов

- 🫀 **CardioLife Index** — персональный индекс здоровья (0–100)
- 📊 **Тест сердца** — 6 вопросов, 2 минуты, мгновенный результат
- 📈 **ЭКГ / ЭхоКГ** — загрузите анализы, доктор проверит и отправит заключение
- 👨‍👩‍👧 **Семейный контроль** — следите за здоровьем родителей
- 💬 **Чат с Dr. Севарой** — ИИ отвечает от имени врача
- 🎓 **Академия сердца** — курсы, анатомия, нормы показателей

### 👩‍⚕️ Для Dr. Севары (`#admin`)

- 📋 **Очередь анализов** — пациенты загружают, доктор проверяет
- 🤖 **ИИ-черновик** — Claude пишет заключение, доктор редактирует
- 📤 **Отправка пациенту** — только после проверки врача
- 🔬 **Инструмент анализа** — офлайн разбор ЭКГ/ЭхоКГ/анализов мгновенно
- 🎓 **Профессиональное обучение** — ЭКГ-тренажёр, ЭхоКГ-атлас, CME-баллы

## 🛠️ Технологии

- **Фронтенд**: HTML5 + CSS3 + Vanilla JS (один файл, без фреймворков)
- **ИИ**: Claude API (claude-sonnet-4-20250514)
- **Стиль**: Dark UI, красный/белый брендинг
- **Деплой**: GitHub Pages / Netlify (статический сайт)

-----

## ⚙️ Настройка для разработки

Файл готов к работе — просто откройте `index.html` в браузере.
